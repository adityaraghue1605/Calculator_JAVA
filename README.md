GitHub README Description
Java Console Calculator
A lightweight, interactive command-line interface (CLI) application built in Java that performs essential arithmetic operations.

🚀 Features
Core Operations: Perform Addition, Subtraction, Multiplication, and Division on two integers.

Statistical Helper: Quickly calculate the average of two numbers.

Interactive Menu: Simple, numbered prompts to guide the user through different operations.

Clean Logic: Implemented using Scanner for real-time user input and structured if-else blocks for operation routing.

🛠️ Technologies Used
Language: Java

Tooling: Java Utility Library (java.util.Scanner)

📖 How It Works
Upon running the program, the user is greeted with a "Namaste" and a list of available operations.

The user selects an operation by entering a number (1-5).

The program prompts for two integers.

The result is calculated and displayed instantly in the console.
